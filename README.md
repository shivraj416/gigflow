# GigFlow

A mini freelance marketplace platform where clients post gigs and freelancers bid on them.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT (HttpOnly Cookies)
- State: Redux Toolkit

## Features
- Authentication (Login/Register)
- Gig CRUD
- Bidding system
- Hire logic (atomic)
- Password recovery
- Protected dashboard
- Modern UI (glassmorphism)
